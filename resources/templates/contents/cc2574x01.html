Persamaan Kuadrat

Buat lo yang lagi atau udah pernah belajar persamaan kuadrat, pernah gak sih lo bingung?
Misalnya , lo bingung apakah?

$$x_{1} + x_{2} = \frac{b}{a}$$





eh, apa gini ya??


$$x_{1} + x_{2} = \frac{c}{a}$$





Tapi, bukannya make tanda minus ya???

Santai men, gak usah bingung. Setelah lo nonton video-video dan ngerjain soal persamaan kuadrat di sini, lo bakalan bisa nurunin formula sendiri. Jadi, gak ada lagi deh yang namanya banyak-banyak ngapalin rumus. Di bab ini lo bakal belajar persamaan kuadrat secara step by step.

Dimulai dari:

  • Definisi persamaan kuadrat dan akar-akar persamaan kuadrat

    • $$ ax^{2} + bx + c = 0, \quad a \neq 0 \\ (x - x_{1}) \ (x - x_{2}) = 0 \\ x = x_{1} \vee x = x_{2} $$







  • Mencari akar dengan cara melengkapkan bentuk kuadrat sempurna

    • $$ x^{2} - ax + b = 0 \\ a^{2} = b + c \\ \Rightarrow (x - (\frac {a}{2}))^{2} - c = 0 \\ \Rightarrow (x - (\frac {a}{2}))^{2} = c \\ \Rightarrow (x - (\frac {a}{2})) = \pm \sqrt{c} \\ â‡’ x_{(1,2)} = (\frac {a}{2}) \pm \sqrt{c} $$













  • Mengenal rumus kecap

    • $$ x_{(1,2)} = \frac{-b \pm \sqrt{b^{2}-4ac}}{2a} $$





  • Menentukan sifat akar-akar, misal

    • Akar-akar kembar

      • $$D = 0$$




    • Akar_akar positif

      • $$\bullet \quad D\geq 0 \\ \bullet x_{1} + x_{2} > 0 \\ \bullet x_{1} * x_{2} > 0 \\ $$







    • Masih banyak lagi

  • Membuat persamaan kuadrat baru, misal

    • $$x^{2} + ax +b = 0 \\ x_{1} = (\alpha) \wedge x_{2} = (\beta) \\ Buat \ persamaan \ baru \ dgn \ akar-akar \ (\alpha)^2 \ \wedge (\alpha)^2 \\ \Rightarrow (x - \alpha) \ (x - \beta) = 0 \\ \Rightarrow x^{2} + (\alpha^2+\beta^2)x +(\alpha*\beta)^{2} = 0 $$
