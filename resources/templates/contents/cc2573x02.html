Sifat-Sifat Logaritma

Note ini cuma berisi rangkuman sifat-sifat atau rumus logaritma SMA aja. Untuk definisi dan ketentuan awalnya ada di note sebelumnya. Langsung aja ya kita bahas.

Sifat pertama

Misalnya gue punya dua persamaan

$$ a^{m} \ = \ b \\ a^{n} \ = \ c \\ $$

Dari eksponen kita tau

$$ a^{m}*a^{n} \ = \ b*c \ = \ a^{m+n} $$

Dari definisi logaritma yang

$$ □^{\triangle} \ = \ a \quad ⇒ \quad \log_□ a \ = \ \triangle $$

Bisa gue otak-atik sifat eksponen diatas menjadi

$$ a^{m+n} \ = \ b*c \ ⇒ \ \log_a b*c \ = \ m+n $$

mengingat

$$ a^{m} \ = \ b \ ⇒ \ \log_a b \ = \ m \\ a^{n} \ = \ c \ ⇒ \ \log_a c \ = \ n \\ $$

Maka

$$ \log_a b*c \ = \ m+n \ = \ \log_a b \ + \log_a c $$

Dapet deh sifat pertama

$$ \log_a b \ + \log_a c \ = \ \log_a b*c $$

Sifat kedua

Kalo sifat pertama kaya begitu, lo bisa nebak dong kalo gue punya persamaan kaya gini, hasilnya apa?

$$ \log_a b \ - \ \log_a c $$

Iya bener, hasilnya

$$ \log_a b \ - \ \log_a c \ = \ \log_a b/c $$

Kalo di eksponen pangkatnya ditambah, kalo di logaritma dikali. kalo di eksponen dikurang, di logaritma dibagi. Jadi, bisa disebut kalo logaritma adalah inversnya eksponen. Kalo lo bisa menguasai eksponen, lo pasti juga bisa logaritma. Gak usah takut sama lambang log-nya.

Sifat lainnya

  • $$ \log_a a^{b} \ = \ b $$
  • $$ a^{\log_a b} \ = \ b $$
  • $$ \log_a b \ = \ \frac{\log_n b}{\log_n a} \\ 0< n < 1 \quad \vee \quad n ≧ 1 $$
  • $$ \log_a b \ = \ \frac{1}{\log_b a}$$
  • $$ \log_a^{c} b^{d} \ = \ \frac{d}{c} * \log_a b $$
